
# Pregnancy Age Calculator

A simple and user-friendly **Pregnancy Age Calculator** built with **Python & PySide6**.  
This application supports **dual language (English & Persian)** and **dual calendar (Jalali & Gregorian)** to help calculate gestational age easily based on either **LMP (Last Menstrual Period)** or **Ultrasound (Sonography)**.

---

## Features

- 🌐 **Bilingual**: Switch between **English** and **Persian (Farsi)**.  
- 📅 **Dual Calendar Support**: Choose between **Jalali** and **Gregorian** dates.  
- 🤰 **LMP Calculation**: Calculate pregnancy age based on the last menstrual period.  
- 🍼 **Sonography Calculation**: Calculate pregnancy age based on ultrasound data.  
- 🖱️ **User-Friendly UI**: Modern interface with clear input fields and buttons.  
- 📆 **Quick Today Button**: Automatically fill in today's date in your chosen calendar.  

---

## Installation

Make sure you have **Python 3.10+** installed.  

Install dependencies:

```bash
pip install -r requirements.txt
```

Clone the repository:

```bash
git clone https://github.com/yourusername/pregnancy-calculator.git
cd pregnancy-calculator
```

Run the application:

```bash
python main.py
```

---

## Usage

1. Launch the application.
2. Choose **LMP** or **Sonography** calculation.
3. Enter the dates in your preferred calendar (Jalali/Gregorian).
4. Press **Calculate** to see the pregnancy age in **weeks and days**.
5. Switch language anytime using the **EN/FA** button.
6. Change calendar anytime using the **Jalali/Gregorian** button.
